Question Class

Attribute:
Question index 		[variable type: double, {enum} this is the index of the question in the database]
QuestionText 		[variable type: string, max 100 char]
Response-type 		[variable type: string, {options}]
Possible-reponses 	[variable type: string, if "NULL"-->open text box response]
FollowUpTo 		[variable type: array-double {enum}, the index of questions that this question is a follow up to]
